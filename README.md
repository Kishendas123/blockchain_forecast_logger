<h1 align="center">₿ Bitcoin Forecast Logger</h1>

<p align="center">
  A forecasting app that predicts the <b>next-day Bitcoin closing price</b> using
  <b>Random Forest</b>, <b>LSTM</b>, and <b>ARIMA</b>, with a local blockchain logging workflow built using
  <b>Hardhat</b> and <b>Solidity</b>.

<h2>📌 Overview</h2>
<p>
  This project combines <b>machine learning</b>, <b>deep learning</b>, and <b>time-series forecasting</b>
  to estimate the next-day BTC closing price in an interactive Streamlit app.
  Users can compare different model approaches, view predicted direction and confidence,
  and explore a blockchain-based forecast logging workflow in the local development version.
</p>
